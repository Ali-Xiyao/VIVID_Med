# Legacy VIVID-Med Archive

Pre-BiVES VIVID-Med code is retained here for provenance only:

- configs, launchers, analysis, training, and evaluation scripts;
- model/training/evaluation packages and old data loaders;
- profiles, prompts, paper source, and evaluator experiments;
- Qwen2.5-era, CT, LP, structured-supervision, and prior baseline routes.

Nothing in this directory is an active experiment entry. Active code lives in
the repository-root `bives_cxr/`, `configs/bives_cxr/`, and the five scripts
listed in the root `README.md`. Active model loading is multimodal Qwen3.5 only.
