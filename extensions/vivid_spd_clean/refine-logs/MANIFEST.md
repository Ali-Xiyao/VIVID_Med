# Manifest

Tracked authority files:

- protocol and JSON lock;
- implementation and tests;
- launch and queue scripts;
- de-identified scalar summaries, hashes, and terminal verdicts when selected
  for publication.

Untracked runtime assets:

- medical images and patient-level manifests;
- model weights and caches;
- checkpoints and generated embeddings;
- environments, credentials, and scheduler output.
