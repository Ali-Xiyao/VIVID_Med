# VIVID extensions

## RCSD-CXR

[`rcsd_cxr/`](rcsd_cxr/) is the clean VIVID journal-extension package for
reliability-calibrated structured semantic distillation.

Its full combination is NO-GO after the prospective G2 and G3 gates. The VIVID
journal extension remains open only for the bounded component-attribution
audit documented under [`rcsd_cxr/audit/`](rcsd_cxr/audit/). Read
[`../VIVID_EXTENSION_BRANCH.md`](../VIVID_EXTENSION_BRANCH.md) before using or
interpreting the package.
