# VIVID-Med 导师汇报

> 最后更新：2026-02-16

## 一句话概述

用冻结 LLM 作为结构化语义监督空间训练 ViT，训练后丢弃 LLM，只交付 ViT backbone。在 CheXpert linear probe 和 NIH 跨域评估上均显著超越 ImageNet baseline。

---

## 1. 论文主实验（已完成）

### 1.1 主表：Linear Probe（⭐ 冻结 ViT，只训练线性分类头）

评估协议：CheXpert 14-label 全量（~29k 训练样本），冻结 ViT backbone，只训练线性分类头。Checkpoint 选择规则：取 val_loss 最低的 step 报告所有指标。

| 实验 | best step | val_loss | macro_F1 | macro_AUC | vs Baseline |
| --- | ---: | ---: | ---: | ---: | --- |
| Baseline (ImageNet) | 2000 | 0.3047 | 0.9004 | 0.6969 | — |
| V10 SAR+HFP | 2600 | 0.2520 | 0.9133 | 0.8137 | F1 +1.29pp, AUC +11.68pp |
| No mask+No cons（纯框架） | 2800 | 0.2584 | 0.9085 | 0.8364 | F1 +0.81pp, AUC +13.95pp |
| **V10.1 SPD** | 2600 | **0.2461** | **0.9248** | 0.8069 | **F1 +2.44pp**, AUC +11.00pp |

配置文件：
- Baseline: `configs/linear_probe_imagenet_full14.yaml`
- V10 SAR+HFP: `configs/linear_probe_v10_sar_hfp_full14.yaml`
- No mask+No cons: `configs/linear_probe_no_mask_no_cons_full14.yaml`
- V10.1 SPD: `configs/linear_probe_v10.1_spd_full14.yaml`

Linear probe 超参数：backbone_lr=0, head_lr=1e-3, weight_decay=0, max_steps=3000, eval_interval=200, drop_path_rate=0.0, batch_size=16, grad_accum=2, bf16=true。

### 1.2 NIH 跨域泛化（8 共同标签，25,596 样本）

用 CheXpert 训练的 linear probe checkpoint 直接在 NIH ChestX-ray14 上测试，不重新训练。

| 实验 | macro_F1 | macro_AUC | vs Baseline |
| --- | ---: | ---: | --- |
| Baseline (ImageNet) | 0.2033 | 0.6159 | — |
| No mask+No cons | 0.2384 | 0.6748 | F1 +3.51pp, AUC +5.89pp |
| V10 SAR+HFP | 0.2539 | 0.6995 | F1 +5.06pp, AUC +8.36pp |
| **V10.1 SPD** | **0.2599** | **0.7121** | **F1 +5.66pp, AUC +9.62pp** |

排序与 CheXpert LP 完全一致（SPD > SAR+HFP > No mask+No cons > Baseline），表征质量提升可跨域泛化。F1 绝对值偏低是预期行为（分类头在 CheXpert 上训练，threshold=0.5 不是 NIH 最优），AUC 不受 threshold 影响，是跨域核心指标。

脚本：`python scripts/eval_nih_crossdomain.py --checkpoint <best.pt> --label <name>`

### 1.3 补充：Full Fine-tune（解冻 ViT，backbone_lr=1e-5）

## 2. 消融实验（已完成 5 组）

### 2.1 消融矩阵

基于 V9 SVRL-full（GFTM + Consistency + CXR+CT 混合），逐一去掉组件观察影响。评估方式为 full fine-tune（14-label 全量，6000 steps）。

| 实验 | GFTM | Cons | CT | macro_F1 | macro_AUC | 状态 |
| --- | --- | --- | --- | ---: | ---: | --- |
| SVRL-full | ✓ | ✓ | ✓ | 0.9064 | 0.8210 | ✅ |
| -GFTM | ✗ | ✓ | ✓ | 0.9208 | 0.8423 | ✅ ⚠️ 反超 |
| -Consistency | ✓ | ✗ | ✓ | 0.9130 | 0.8439 | ✅ ⚠️ 反超 |
| No mask+No cons | ✗ | ✗ | ✗ | 0.9152 | 0.8563 | ✅ ⚠️ AUC 最高 |
| -CT (CXR only) | ✓ | ✓ | ✗ | 0.9149 | 0.8446 | ✅ 反超 |
| Random mask (VRL) | random 75% | ✓ | ✓ | — | — | ⬜ 待跑 |
| ImageNet baseline | — | — | — | 0.8987 | 0.7927 | ✅ |

配置文件：
- SVRL-full: `configs/v9_svrl_cxr_ct.yaml`
- -GFTM: `configs/ablation_v9_no_gftm.yaml`
- -Consistency: `configs/ablation_v9_no_consistency.yaml`
- No mask+No cons: `configs/ablation_v9_no_mask_no_consistency.yaml`
- -CT: `configs/ablation_v9_cxr_only.yaml`

### 2.2 消融关键发现

1. **所有消融均优于 SVRL-full** → GFTM、Consistency、CT 混合三个组件均无正面贡献
2. **No mask+No cons（纯框架）AUC 最高 0.8563** → 核心价值来自 frozen LLM supervision 框架本身
3. **GFTM 是预训练过拟合主因**：含 GFTM 的实验 best@500-1000 后退化，不含 GFTM 的训练健康（best@5000-6500）
4. **混合 CT 反而拖累 CXR transfer**：-CT（仅 CXR）F1/AUC 均优于 SVRL-full（CXR+CT），CMSA novelty 不成立
5. 原方案"三个 novelty（GFTM + FSA + CMSA）"已不成立，核心贡献重新定位为 Frozen LLM Supervision 框架

## 3. V10/V10.1 新模块（基于消融结论设计）

消融结论：纯框架最优，GFTM hard masking 有害。V10/V10.1 在纯框架基础上加轻量级增强模块。

### 3.1 V10 SAR+HFP

- **SAR (Soft Attention Reweighting)**：用 CLS-to-patch attention 倒数作为软权重，放大低注意力 tokens，替代 GFTM 的 hard masking。可学习参数 α 控制加权强度。
- **HFP (Hierarchical Feature Projection)**：提取 ViT 第 4/8/12 层特征，每层一个 MLP projector → concat → fusion，让梯度传到 ViT 所有层。
- 预训练配置：`configs/v10_sar_hfp.yaml`（10k steps, CXR+CT）
- Linear probe 结果：F1=0.9133, AUC=0.8137（vs Baseline +1.29pp/+11.68pp）

### 3.2 V10.1 SPD（当前最优）

- **SPD (Structured Prediction Decomposition)**：3 组可学习 query tokens（每组 2 个），通过 cross-attention 从 ViT 特征中自适应提取不同方面信息。正交性损失（ortho_weight=0.01）鼓励各组 attention pattern 互不重叠。
- 与 HFP 的区别：HFP 纵向多尺度（多层融合）→ SPD 横向多方面（单层分解）
- 预训练配置：`configs/v10.1_spd.yaml`（10k steps, CXR+CT）
- Linear probe 结果：**F1=0.9248**（所有实验最高），AUC=0.8069
- NIH 跨域：**F1=0.2599, AUC=0.7121**（所有实验最高）

## 6. 论文叙事方向

- **核心贡献**：Frozen LLM Supervision for Visual Representation Learning — 用冻结 LLM 作为结构化语义监督空间，训练后丢弃 LLM，只交付 ViT backbone
- **主表**：Linear Probe（CheXpert 14-label）+ NIH 跨域泛化
- **方法对比**：ImageNet baseline / No mask+No cons（纯框架）/ V10 SAR+HFP / V10.1 SPD
- **消融**：证明简单框架优于复杂组合（GFTM/Consistency/CT 混合均无正面贡献）
- **与 ViTP 的差异**：冻结 LLM（vs 训练 LLM）、SPD 分解（vs random token drop）、结构化 JSON 预测（vs caption generation）
- CMSA 作为 negative finding 报告，不作为 contribution
