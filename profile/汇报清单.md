# VIVID-Med 导师汇报

> 最后更新：2026-02-21（V12 重构版，SPD only，全部实验已完成）

## 一句话概述

用冻结 LLM 作为结构化语义监督空间训练 ViT，训练后丢弃 LLM，只交付 ViT backbone。主方法 = Frozen LLM + UMS 结构化 JSON + SPD 分解式投影 + 12-label 预训练。

---

## 1. 主方法

SPD(3组×2tokens, ortho=0.01) + 12-label + field_query_training + answerability_mask

```text
Image → ViT(trainable) → vit_features
      → SPD Projector: 多组 query tokens × cross-attention → SPD tokens → 共享MLP投影
      → Frozen LLM (Qwen2.5-1.5B) → UMS-JSON tokens
```

> HFP（多层融合）已排除：历史数据显示 SPD+HFP 反而低于 SPD 单独（LP AUC 0.7930 vs 0.8441）。
> 历史数据（V9/V10/V10.1/V11 及旧消融结果）已归档至 `profile/历史数据.md`。
