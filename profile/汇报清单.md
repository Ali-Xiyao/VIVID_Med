## 阶段一：生成链路探索（V4-V7）

### 核心问题

早期尝试直接用生成链路（ViT→Projector→LLM→JSON）的输出质量与分类 baseline 对比，发现生成链路 macro_f1 远低于纯分类 baseline（0.5115 vs 0.8839），差距 42%。

### V4-V6 关键结论

- V4：14 字段全空塌缩（pred_nan_rate=1.0），模型只输出 null
- V4.2：dense10k 采样后部分恢复（macro_f1=0.2553），但长尾字段仍失效
- V5/V5.1：Top3 字段聚焦证明头部字段可学（macro_f1=0.8909），但字段间不平衡
- V6：Top7 双线并行，脱离全空塌缩（macro_f1=0.4087），abstain 口径虚高不可用

### V7 关键结论

- V7 主线 macro_f1=0.5115，仍远低于 baseline 0.8839
- V7.1 低召回修复过激导致退化（macro_f1=0.4617）
- **策略转向**：放弃"生成链路直接超越 baseline"，聚焦 ViT-only transfer

### 同头公平对比（V7 时期建立）

首次建立"同头公平对比"协议：两边都用 ViT + 同构分类头(BCE)，仅改变 ViT 初始化来源。

| 实验 | macro_f1 | micro_f1 |
| --- | ---: | ---: |
| Baseline (ImageNet init) | 0.8839 | 0.8974 |
| From-VIVID V7 | 0.8857 | 0.8984 |
| 差值 | +0.0018 | +0.0011 |

Route B（分组学习率）提升更明显：macro_f1 +0.0143。

## 阶段二：Transfer 协议与架构优化（V8）

### V8 改动

| 项目 | V7 | V8 | 理由 |
| --- | --- | --- | --- |
| vit_output_type | cls (1 token) | all (197 tokens) | 迫使 ViT 在 patch 级别编码 |
| num_prefix_tokens | 16 | 4 | 减少 Projector 免费容量 |
| projector_mlp_hidden_dim | 3072 | 1536 | 削弱 Projector |
| 学习率 | 统一 8e-5 | ViT=2e-5, Proj=1e-4 | 差异化学习率 |
| max_steps | 3500 | 5000 | 更多有效梯度 |

### V8 结果

| 实验 | macro_f1 | vs Baseline |
| --- | ---: | --- |
| Baseline (ImageNet) | 0.8839 | — |
| From-VIVID V8 (best) | 0.8852 | +0.0013 |

结论：4 项改动未带来预期提升，根本原因是训练数据量不足（仅 10k CXR）、缺少迫使 ViT 学习的机制。

## 阶段三：V9 SVRL 完整实现与消融（2026-02-13 ~ 02-14）

### V9 设计

方法名：SVRL — Structured Visual Representation Learning via Frozen LLM Supervision

三个组件：

1. GFTM (Gradient-Focused Token Masking)：attention-guided token masking
2. FSA Consistency：冻结 LLM 作为稳定锚点的 augmentation consistency loss
3. CMSA：CXR + CT 统一 schema 混合训练（7:3）

数据：CXR 29k（CheXpert）+ CT 26k slices（AMOS 2.5D），10000 steps 预训练。

### V9 预训练结果

best_val_loss=0.2901 @ step 1000（后续退化，GFTM 导致过拟合）

### V9 Transfer 结果（14-label 全量口径）

| 实验 | macro_F1 | macro_AUC |
| --- | ---: | ---: |
| Baseline (ImageNet) | 0.8987 | 0.7927 |
| SVRL (7-label pretrain) | 0.9064 | 0.8210 |
| SVRL (14-label pretrain) | 0.9144 | 0.8261 |

**首次在 14-label 全量口径下全面超越 baseline**，AUC +2.83pp。

### V9 消融实验（关键转折点）

| 实验 | GFTM | Consistency | CT | macro_F1 | macro_AUC |
| --- | --- | --- | --- | ---: | ---: |
| SVRL-full | ✓ | ✓ | ✓ | 0.9064 | 0.8210 |
| -GFTM | ✗ | ✓ | ✓ | **0.9208** | 0.8423 |
| -Consistency | ✓ | ✗ | ✓ | 0.9130 | 0.8439 |
| No mask+No cons | ✗ | ✗ | ✗ | 0.9152 | **0.8563** |
| -CT (CXR only) | ✓ | ✓ | ✗ | 0.9149 | 0.8446 |
| ImageNet baseline | — | — | — | 0.8987 | 0.7927 |

**核心发现**：

- 所有消融均优于 SVRL-full → 三个组件均为 net negative
- 纯框架（No mask+No cons）AUC=0.8563，比 baseline +6.4pp → 核心价值来自框架本身
- GFTM 是过拟合主因：含 GFTM best@500-1000，不含 GFTM best@5000-6500
- 混合 CT 拖累 CXR transfer：-CT 优于 SVRL-full（F1 +0.85pp, AUC +2.36pp）

## 阶段四：V10/V10.1 新模块设计（2026-02-14 ~ 02-15）

### 论文贡献重新定位

基于消融结论，放弃 GFTM/FSA/CMSA 作为 novelty，核心贡献重新定位为：

1. **Frozen LLM Supervision 框架本身**（消融证明纯框架最优）
2. **新增模块**（V10 SAR+HFP 或 V10.1 SPD，待实验验证）
3. **评估协议升级**（linear probe + low-data + cross-domain）

### V10 SAR + HFP

- SAR：软加权替代 GFTM hard masking，不丢弃 token
- HFP：多层 ViT 特征融合（第 4/8/12 层）
- 代码已实现，待跑预训练

### V10.1 SPD

- 多组 cross-attention query tokens 分解视觉表征
- 正交性损失鼓励各组关注不同区域
- 代码已实现，待跑预训练

### 评估协议升级

- Linear probe（主表）：冻结 ViT，只训练线性头，直接衡量表征质量
- Low-data（10%/25%/50%）：数据效率曲线
- Cross-domain（NIH ChestX-ray14）：跨机构泛化
- Full fine-tune：补充（当前方式）
- CMSA 不作为 contribution，作为 negative finding 报告
